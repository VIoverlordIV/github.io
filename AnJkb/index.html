🟢 ジャンケンアプリ（Android）

Android Studioで開発したシンプルなジャンケンゲームアプリです。
ゲームの基本ルールは「グー・チョキ・パー」で勝負するだけという簡単な構成ですが、 アニメキャラクターを導入することで全体のビジュアル体験を高めました。

キャラクターは勝敗に応じて「喜・怒・哀」の表情を変化させるように設計しました。 これにより、シンプルなミニゲームながらも感情的な楽しさを感じられる作品になっています。

最も苦労した点は、GIFアニメーションをAndroid上で自然に動かす実装部分です。 通常の画像表示では動作しないため、外部ライブラリを活用して動作を最適化する工夫を行いました。

  • 🟢 開発環境：Android Studio
  • 🟢 使用言語：Kotlin / Java
  • 🟢 主要機能：キャラの表情変化・GIFアニメ表示
